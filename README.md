# SOC Detection and Log Analysis

This repository contains hands-on Security Operations Center (SOC) labs focused on log analysis, event investigation, process review, and detection fundamentals.

## Objectives

* Develop familiarity with security event logs.
* Learn common SOC investigation workflows.
* Practice reviewing Windows security events.
* Build foundational incident response skills.
* Document investigative findings using analyst-style reporting.

## Labs

### Lab 01 — Failed Logon Investigation
Review authentication events and identify failed login attempts.

### Lab 02 — Account Lockout Analysis
Investigate account lockout activity and determine likely causes.

### Lab 03 — Suspicious PowerShell Activity
Analyze PowerShell execution activity and review indicators of suspicious behavior.

### Lab 04 — Windows Event Viewer Basics
Review Windows event logs and identify relevant security events.

### Lab 05 — Process Investigation
Investigate running processes and document findings.

## Disclaimer

These labs were performed in a personal lab environment for educational and portfolio development purposes.
